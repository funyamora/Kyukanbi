# ホームボタン控えめ化・リマインダー時刻追加・Androidタブアイコン修正 - 2026年03月14日 08:00

## 作業内容
4つのUI改善・バグ修正を実施。ホーム画面の「今日は飲む」ボタンを控えめなスタイルに変更、オンボーディングと設定画面のリマインダー時刻選択肢を追加、Android実機でタブアイコンが表示されない問題を修正。

## 変更詳細

### 1. ホーム画面「今日は飲む」ボタンの控えめ化
オレンジのべた塗りボタン（btnPrimary）から、控えめなボーダースタイル（btnDrink）に変更。
飲酒を促すようなデザインを避け、休肝日を選択しやすくする意図。

```tsx
// 変更前
style={({ pressed }) => [styles.btnPrimary, { backgroundColor: "#FF6B35" }, pressed && { opacity: 0.85 }]}
<Text style={styles.btnPrimaryText}>🍺 今日は飲む（上限を宣言する）</Text>

// 変更後
style={({ pressed }) => [styles.btnDrink, { borderColor: colors.muted }, pressed && { opacity: 0.6 }]}
<Text style={[styles.btnDrinkText, { color: colors.muted }]}>🍺 今日は飲む（上限を宣言する）</Text>
```

### 2. オンボーディングのリマインダー時刻に18:00と23:00を追加
```tsx
// 変更前

// 変更後
const TIME_OPTIONS = ["18:00", "19:00", "20:00", "21:00", "22:00", "23:00"];
```

### 3. 設定画面のリマインダー時刻に23:00を追加
```tsx
// 変更前
const REMINDER_TIME_OPTIONS = ["18:00", "19:00", "20:00", "21:00", "22:00"];

// 変更後
const REMINDER_TIME_OPTIONS = ["18:00", "19:00", "20:00", "21:00", "22:00", "23:00"];
```

### 4. Androidタブアイコン非表示の修正
`expo-asset` が v55.0.8（SDK 55用）にインストールされていたが、アプリは SDK 54。
`expo-asset` v55 は `expo-modules-core` の `getFilePermission()` を呼ぶが SDK 54 にはこのメソッドが存在せず、
`Font.loadAsync` → `ExpoAsset.downloadAsync` が失敗 → MaterialIcons フォントがロードできなかった。
`npx expo install expo-asset` で SDK 54 対応の v12.0.12 にダウングレードして解決。
