# 週間セルタップ遷移と飲まないボタン強化 - 2026年03月14日 08:20

## 作業内容
ホーム画面の「今週の休肝日」の各曜日セルをタップ可能にし、適切な画面への遷移を実装。
また「今日は飲まない」ボタンをアウトラインから塗りつぶしスタイルに変更して存在感を強化。

## 変更ファイル
- app/(tabs)/index.tsx
- app/(tabs)/record.tsx
- doc/画面設計書.md

## 変更詳細

### 週間セルをタップ可能に（index.tsx）
- `weekDates.map()` 内の `<View>` を `<Pressable>` に変更
- 過去・今日の日付タップ → 記録タブへ日付パラメータ付き遷移
- 未来の日付タップ → 週間計画タブへ遷移
- タップ時にハプティクスフィードバック（Light）を追加

### 記録画面で日付パラメータ受け取り（record.tsx）
- `useLocalSearchParams` で `date` パラメータを取得
- `selectedDate` の初期値に使用
- `useEffect` でパラメータ変更時に `selectedDate` を更新

### 「今日は飲まない」ボタンを塗りつぶしに（index.tsx）
- `btnSecondary`（アウトライン）→ `btnNodrink`（塗りつぶし `#4A90D9`）に変更
- 設定済み状態も `btnNodrinkDone`（塗りつぶし `#4CAF50`）に統一
- テキスト白、fontSize: 18、padding: 18 で存在感を強化

### 変更箇所（行番号）
```typescript
// index.tsx: View → Pressable（週間セル）
// 変更前
<View key={date} style={styles.dayCell}>

// 変更後
<Pressable
  key={date}
  style={({ pressed }) => [styles.dayCell, pressed && { opacity: 0.6 }]}
  onPress={() => { /* 遷移ロジック */ }}
>

// record.tsx: 日付パラメータ受け取り追加
const params = useLocalSearchParams<{ date?: string }>();
const [selectedDate, setSelectedDate] = useState(params.date ?? today);

// index.tsx: ボタンスタイル変更
// 変更前
btnSecondary: { borderRadius: 14, padding: 14, borderWidth: 2, borderColor: "#4A90D9", backgroundColor: "transparent" }

// 変更後
btnNodrink: { borderRadius: 14, padding: 18, backgroundColor: "#4A90D9" }
btnNodrinkDone: { borderRadius: 14, padding: 18, backgroundColor: "#4CAF50" }
btnNodrinkText: { fontSize: 18, fontWeight: "700", color: "#fff" }
```
