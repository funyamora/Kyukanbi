# 週表示ラベルの短縮形式統一 - 2026年03月14日 12:10

## 作業内容
週間計画画面と振り返り画面の週表示ラベルを、より自然で簡潔な形式に改善。

## 変更ファイル
- app/(tabs)/weekly.tsx
- app/(tabs)/review.tsx

## 変更詳細

### 1. 週ナビゲーションラベルの改善（weekly.tsx）
`+1週`/`-1週` という不自然な表示を、`今週`/`来週`/`先週` に変更。
2週以上離れた場合は `3/9(月)〜3/15(日)` 形式で表示。

```typescript
// 変更前
weekOffset === 0 ? "今週" : weekOffset > 0 ? `+${weekOffset}週` : `${weekOffset}週`

// 変更後
weekOffset === 0 → "今週"
weekOffset === 1 → "来週"
weekOffset === -1 → "先週"
それ以外 → "3/9(月)〜3/15(日)" 形式
```

### 2. ヘッダーサブタイトルの短縮（weekly.tsx）
```typescript
// 変更前
"3月9日（月）〜 3月15日（日）"

// 変更後
"3/9(月)〜3/15(日)"
```

### 3. 振り返り画面の週ラベル短縮（review.tsx）
```typescript
// 変更前
"3月9日（月）〜 3月15日（日）"

// 変更後
"3/9(月)〜3/15(日)"
```

## 影響範囲
- 週間計画画面のヘッダーと週ナビゲーション表示
- 振り返り画面の週タブでの期間表示

## テスト項目
1. 週間計画画面で今週/来週/先週が正しく表示されること
2. 2週以上前後に移動した場合に日付範囲形式で表示されること
3. 振り返り画面の週タブで短縮形式が正しく表示されること
4. 月タブの表示には影響がないこと
