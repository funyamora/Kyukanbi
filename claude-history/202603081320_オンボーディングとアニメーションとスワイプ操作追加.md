### 機能1: オンボーディング画面

`app/onboarding.tsx` を新規作成。3ステップのフロー:
- Step 0: アプリ紹介（🍵 + 説明文）
- Step 1: 目標設定（weeklyGoalDays ステッパー + requireConsecutive スイッチ）
- Step 2: 通知設定（reminderEnabled スイッチ + 時刻選択チップ）

`app/_layout.tsx` に `OnboardingGate` コンポーネントを追加:
```tsx
// 追加（43行目付近）
function OnboardingGate() {
  const router = useRouter();
  useEffect(() => {
    AsyncStorage.getItem("onboarding_completed").then((val) => {
      if (!val) router.replace("/onboarding");
    });
  }, [router]);
  return null;
}
```

Stack.Screen に onboarding を登録:
```tsx
<Stack.Screen name="onboarding" options={{ presentation: "fullScreenModal", headerShown: false }} />
```

### 機能2: ホーム画面アニメーション

`app/(tabs)/index.tsx` に reanimated アニメーションを追加:
- ステータス変更時: カードのスケールパルス（1.0→0.97→1.0、各75ms）
- 2連続達成時: 緑オーバーレイフラッシュ（opacity 0→1→0、各150ms）
- ハプティクス: kyukan選択で Light、達成で Success

```tsx
// 変更前
<View style={[styles.todayCard, { backgroundColor: todayCardBg }]}>

// 変更後
<Animated.View style={[styles.todayCard, { backgroundColor: todayCardBg }, cardAnimatedStyle]}>
  <Animated.View pointerEvents="none" style={[StyleSheet.absoluteFill, { backgroundColor: "#4CAF50", borderRadius: 20 }, overlayAnimatedStyle]} />
```

### 機能3: 週間スワイプ操作

`app/(tabs)/weekly.tsx` に `SwipeableDayRow` コンポーネントを追加:
- Gesture.Pan() + Gesture.Tap() を Gesture.Race() で合成
- 右スワイプ→休肝日、左スワイプ→飲酒OK（閾値60px、クランプ80px）
- failOffsetY([-10, 10]) で FlatList 縦スクロールと競合回避
- スワイプ中に背景色プレビュー（緑/オレンジ）を opacity interpolate で表示

```tsx
// 変更前: Pressable ベースの renderItem
<Pressable style={...} onPress={() => handleToggle(date)}>

// 変更後: SwipeableDayRow コンポーネント
<SwipeableDayRow date={date} isToday={isToday} rec={rec} colors={colors}
  onSetKyukan={handleSetKyukan} onSetOk={handleSetOk} onTap={handleToggle} />
```
