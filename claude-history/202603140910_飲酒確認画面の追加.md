# 飲酒確認画面の追加 - 2026年03月14日 09:10

## 作業内容
ホーム画面の「今日は飲む」ボタンを押した際に、すぐに宣言画面に遷移するのではなく、飲酒を思いとどまらせる中間確認画面（「本当に飲みますか？」）を挟むように変更。同時にホーム画面の「代替行動を見る」ボタンは確認画面に移動したため削除。

## 変更ファイル
- `app/confirm-drink.tsx`（新規作成）
- `app/(tabs)/index.tsx`（修正）
- `app/_layout.tsx`（修正）
- `app.config.ts`（修正）
- `CLAUDE.md`（修正）
- `doc/画面設計書.md`（修正）
- `doc/開発ガイド.md`（修正）

## 変更詳細

### 飲酒確認画面の新規作成（app/confirm-drink.tsx）
- 「本当に飲みますか？」の大きなテキスト表示
- 厚労省・e-ヘルスネットの飲酒抑止豆知識をランダム表示（TRIVIA_LINKS配列、5件）
- 「詳しく見る」ボタンでブラウザ起動（Linking.openURL）
- 3つのアクションボタン:
  - 「✅ やっぱり飲まない」→ 休肝日記録 + router.back()
  - 「💡 替わりの行動で気を紛らわす」→ router.replace("/alternative")
  - 「🍺 ルールを決めてから飲む」→ router.replace("/declaration")

### ホーム画面の修正（app/(tabs)/index.tsx）
- `handleDrink`の遷移先を `/declaration` → `/confirm-drink` に変更
- `handleAlternative`コールバックと「代替行動を見る」ボタンを削除
- 未使用スタイル（btnRow, btnGhost, btnGhostText）を削除

### ルートレイアウトの修正（app/_layout.tsx）
- `confirm-drink`をモーダルルートとして追加

### Web出力設定の修正（app.config.ts）
- `web.output`を`"static"`→`"single"`に変更（NativeWindのSSRエラー回避）

### ドキュメント更新
- CLAUDE.md: ファイル構成にconfirm-drink.tsx追加、豆知識リンク管理セクション新設
- 画面設計書.md: モーダル一覧・画面遷移図・ホーム画面アクションを更新
- 開発ガイド.md: ファイル構成にconfirm-drink.tsx追加

### 変更箇所（行番号）
```typescript
// app/(tabs)/index.tsx - 遷移先変更
// 変更前
router.push("/declaration");

// 変更後
router.push("/confirm-drink");
```

## 影響範囲
- ホーム画面の飲酒ボタンの遷移フロー全体
- Web版の出力方式（static → single）
