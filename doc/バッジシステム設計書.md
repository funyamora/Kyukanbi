# バッジシステム設計書：休肝日つくーる

## 1. 概要

バッジシステムは、ユーザーの継続的な休肝日習慣をモチベートするためのゲーミフィケーション機能である。特定の条件を達成すると自動的にバッジが解除され、ホーム画面と振り返り画面に表示される。

## 2. バッジ定義

`lib/store.ts` の `BADGE_DEFINITIONS` で定義される5種類のバッジがある。

| ID | 絵文字 | 名前 | 解除条件 |
| :--- | :--- | :--- | :--- |
| `first_kyukan` | 🥉 | 初めての休肝日 | 初めて `status: "kyukan"` の記録が作成されたとき |
| `first_consecutive` | 🥈 | 初めての2連続 | 初めて2日連続の休肝日を達成したとき |
| `first_weekly_goal` | 🥇 | 週間目標達成 | 週の休肝日数が `weeklyGoalDays` 以上になったとき |
| `three_weeks_streak` | 💎 | 3週連続達成 | 3週間連続で週間目標を達成したとき |
| `one_month_streak` | 🌟 | 1ヶ月継続 | 4週間（28日間）連続で週間目標を達成したとき |

## 3. 解除判定ロジック

### checkNewBadges()

```typescript
function checkNewBadges(
  records: Record<string, DailyRecord>,
  existingBadges: string[],
  weeklyGoalDays: number
): string[]
```

- **引数**:
  - `records`: 全日次記録
  - `existingBadges`: すでに解除済みのバッジID配列
  - `weeklyGoalDays`: 週の休肝日目標日数
- **戻り値**: 新たに解除されたバッジIDの配列（すでに解除済みのものは含まない）
- **呼び出しタイミング**: `patchRecord()` 実行後に自動的に呼び出される

### 判定の流れ

1. `existingBadges` に含まれないバッジのみを判定対象とする
2. 各バッジの条件を `records` に対して評価する
3. 新たに条件を満たしたバッジIDを配列で返す
4. 呼び出し元（`patchRecord`）が `AppStore.badges` に追加し、`saveStore()` で永続化する

## 4. データの保存

バッジデータは `AppStore.badges` に `string[]` として保存される。

```typescript
interface AppStore {
  records: Record<string, DailyRecord>;
  badges: string[];  // 例: ["first_kyukan", "first_consecutive"]
}
```

AsyncStorageキー `kyukoubi_store_v1` 内に含まれる。

## 5. UI表示

### BadgeList コンポーネント

`components/BadgeList.tsx` に定義された再利用可能なコンポーネント。

```typescript
interface BadgeListProps {
  earnedBadges: string[];
}

function BadgeList({ earnedBadges }: BadgeListProps)
```

- **表示形式**: 横スクロールの `FlatList`
- **バッジ表示**: `BADGE_DEFINITIONS` の全5バッジを常に表示
  - **解除済み**: 絵文字 + 名前を通常表示。`colors.primary` の背景円
  - **未解除**: 不透明度 0.3 でグレーアウト表示
- **使用箇所**: ホーム画面（`index.tsx`）、振り返り画面（`review.tsx`）

## 6. 通知連携

バッジ解除とは別に、2連続休肝日達成時には `sendAchievementNotification()` が呼び出される。

- **条件**: `patchRecord` で `status: "kyukan"` を設定した際、`settings.achievementNotification` が `true` かつ `checkConsecutiveAchievement()` が `true` のとき
- **通知内容**: 「🎉 2連続休肝日達成！」
