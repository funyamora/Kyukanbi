# コンポーネント一覧：休肝日つくーる

本ドキュメントでは、アプリ内で使用される再利用可能なコンポーネントの一覧とその仕様を記載する。

---

## ScreenContainer

**ファイル**: `components/screen-container.tsx`

全画面で使用するSafeAreaラッパー。画面背景色の設定とSafeAreaの管理を行う。

### Props

| Prop | 型 | デフォルト | 説明 |
| :--- | :--- | :--- | :--- |
| `edges` | `Edge[]` | `["top", "left", "right"]` | SafeAreaのエッジ指定 |
| `className` | `string` | - | コンテンツViewのNativeWindクラス |
| `containerClassName` | `string` | - | 最外層View（背景）のクラス |
| `safeAreaClassName` | `string` | - | SafeAreaViewのクラス |
| `style` | `StyleProp<ViewStyle>` | - | スタイル |
| `children` | `ReactNode` | - | 子要素 |

### 使用例

```typescript
import { ScreenContainer } from "@/components/screen-container";

export default function MyScreen() {
  return (
    <ScreenContainer className="p-4">
      {/* 画面の内容 */}
    </ScreenContainer>
  );
}
```

### 注意事項

- **全画面で必ず使用すること。** SafeAreaの管理とテーマ背景色の適用を担う。

## BadgeList

**ファイル**: `components/BadgeList.tsx`

バッジの一覧を横スクロールで表示するコンポーネント。

### Props

| Prop | 型 | 説明 |
| :--- | :--- | :--- |
| `earnedBadges` | `string[]` | 解除済みバッジIDの配列 |

### 使用例

```typescript
import { BadgeList } from "@/components/BadgeList";

<BadgeList earnedBadges={store.badges} />
```

### 表示仕様

- `BADGE_DEFINITIONS` の全5バッジを常に表示
- 解除済みバッジ: 通常表示（`colors.primary` 背景円 + 絵文字 + 名前）
- 未解除バッジ: 不透明度 0.3 でグレーアウト
- 横スクロール `FlatList` を使用

## WeekdayChart

**ファイル**: `components/WeekdayChart.tsx`

曜日別の平均飲酒杯数を棒グラフで表示するSVGコンポーネント。

### Props

| Prop | 型 | 説明 |
| :--- | :--- | :--- |
| `data` | `{ dayIndex: number; label: string; avg: number }[]` | 曜日別平均データ |

### 使用例

```typescript
import { WeekdayChart } from "@/components/WeekdayChart";
import { computeWeekdayAverages } from "@/lib/store";

const data = computeWeekdayAverages(store.records);
<WeekdayChart data={data} />
```

### 表示仕様

- `react-native-svg` の `Rect`, `Text` を使用した棒グラフ
- チャート高さ: 120px
- 最大飲酒量の曜日: `colors.orange` でハイライト
- その他の曜日: `colors.primary`
- バーの上に平均値を表示
- チャート下部にピーク曜日のコメントを表示

## HapticTab

**ファイル**: `components/haptic-tab.tsx`

タブバーのボタンとして使用される、ハプティクスフィードバック付きのタブボタン。

### 仕様

- タブタップ時に `Haptics.impactAsync(Light)` を発火
- `Platform.OS !== "web"` のときのみハプティクスを実行
- `app/(tabs)/_layout.tsx` で `tabBarButton` として設定済み

---

## IconSymbol

**ファイル**: `components/ui/icon-symbol.tsx`（全プラットフォーム）、`components/ui/icon-symbol.ios.tsx`（iOS専用）

SF Symbols風の名前からMaterialIconsへのマッピングを行うアイコンコンポーネント。

### Props

| Prop | 型 | デフォルト | 説明 |
| :--- | :--- | :--- | :--- |
| `name` | `IconSymbolName` | - | SF Symbols風のアイコン名 |
| `size` | `number` | `24` | アイコンサイズ |
| `color` | `string \| OpaqueColorValue` | - | アイコン色 |
| `style` | `StyleProp<TextStyle>` | - | スタイル |

### 注意事項

- 新しいアイコンを使用する前に、必ず `MAPPING` オブジェクトにマッピングを追加すること
- マッピングなしで使用するとアプリがクラッシュする
