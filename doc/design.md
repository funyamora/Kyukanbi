# 休肝日つくーる — デザイン設計書

## スクリーンリスト
1. ホーム画面 — 今日の判定・週の進捗・行動ボタン
2. 週間計画画面 — 曜日別の休肝日/飲酒OK/未定設定
3. 飲酒前宣言画面 — 上限杯数・理由の宣言（モーダルシート）
4. 代替行動画面 — 気分別の代替行動提案・5分タイマー（モーダルシート）
5. 記録画面 — 飲酒実績の入力・宣言との比較
6. 振り返り画面 — 週次/月次の統計・グラフ・インサイト

## カラーパレット
- Primary Blue: #4A90D9（メインアクション・ホームカード）
- Warm Orange: #FF6B35（飲酒前宣言・警告アクション）
- Calm Purple: #667EEA（代替行動）
- Success Green: #4CAF50（休肝日・達成）
- Warning Amber: #F0A500（未達成警告）
- Error Red: #EF4444（上限オーバー）
- Background: #F5F5F7（ライト）/ #1C1C1E（ダーク）
- Surface: #FFFFFF（ライト）/ #2C2C2E（ダーク）

## キーユーザーフロー
1. 通常日: ホーム → 今日の判定確認 → 飲まない宣言
2. 飲む日: ホーム → 「今日は飲む」 → 飲酒前宣言 → 記録
3. 迷った日: ホーム → 代替行動 → タイマー → 飲まない/飲む判断
4. 週初め: 週間計画 → 休肝日を仮置き → 2連続確認

## デザイン方針
- 「今日の判定」を最上部に大きく表示し、迷いを排除する
- 医療アプリ感を避け、親しみやすいトーンを維持する
- ユーザーを責めない。事実を淡々と伝える文言を使う
- タブバーは4タブ（ホーム・週間計画・記録・振り返り）
